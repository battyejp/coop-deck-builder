# Coop Deck Builder

A cooperative deck-building game project built in Unity.

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/battyejp/coop-deck-builder.git
```

### 2. Open the project in Unity
- Open Unity Hub.
- Click **Add** and select the cloned `coop-deck-builder` folder.
- Make sure to use **Unity 2021.3 LTS or above** for best compatibility.

### 3. Run the Project
- In Unity Hub, select the project and click **Open**.
- Go to the **Scenes** folder and open the main scene (e.g., `MainScene.unity`).
- Click the **Play** button at the top of the Unity Editor to run the project.

### 4. Folder Structure  
See `docs/UnityProjectStructure.md` for more details.

### 5. Scripts  
- Place provided scripts into `Assets/Scripts` in your Unity project if not already present.

### 6. Scene Setup  
- Follow `docs/SceneSetup.md` to set up the main scene if it is not already created.
